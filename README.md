# EchoVerse

A spoken word and poetry platform — where every voice finds its stage.

## Features

- Daily spotlight with animated gold microphone hero
- 8 curated spoken word performances embedded from YouTube
- Category filtering (Love, Reflection, Motivation, Identity, Growth, Social Justice)
- Trending artists section
- Upcoming events listings
- Daily affirmation card
- Sidebar navigation with dark/light theme toggle
- Audio player with visual effects
- Luxury card design pattern (Valora-style dark headers with italic serif titles, Save/Share actions)

## Stack

Pure HTML/CSS/JS — no build step required. Open `EchoVerse.html` directly in a browser or deploy as a static site.

## Live

Deployed via [Vercel](https://vercel.com).
